# balatro_save_editor

Just a fairly simple python script to modify your balatro save file.

`balatro_save_editor` contains the main. 

`balatro_save_file` contains the save file editor.

`token_iterator` is an iterator class which removes empty tokens from being returned.

## Save file structure

The save file is compressed with the `deflate` algorithm, which can be done in python with the `zlib` library using `wbits=-zlib.MAX_WBITS`.

Fairly simple to modify with other changes, but made much easier with a debugger/IDE (e.g. Pycharm)

### Edit 'dollars'
                                                  
Sets your in-game money to a specified value.

`balatro_save_editor.edit_money(new_value=19999)`

### Edit 'chips'

Sets your in-game score to 1 less than the target required score to win the ante. In otherwords, playing any hand should net you a win.

### Edit 'multipliers'

Sets the multipliers for each hand type to the specified value.
                                                               
### Edit 'card abilities'

This is useful for setting or removing abilities from cards. I've only used it to remove 'eternal' from jokers.

### Edit 'card limits'

Set the limits for various card areas, like number of jokers, or hand limt.

## Usage

Ensure the path to the save file is correct. If your save file is in `%APPDATA%\Balatro\1\save.jkr` then you won't need to change the path in the script. This is the default for Windows/Steam installations.

Modify the script with the changes you want made, and then **with baltaro closed**, run the script.

## Safety

Backups will be made into the same directory as the save file. The backup is created before writing to the save file.

Changes won't be made unless the script is certain that the save file has been deserialized correctly (and can reserialize to the same original save file.)
